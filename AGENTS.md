# AGENTS

## Project Summary
Company X Storybook is a minimal React + TypeScript + Vite + Tailwind project focused on documenting a fictional website application design system in Storybook.

## What Is Included
- Brand book story with logo, colors, typography, and voice principles.
- Core website-style UI components:
  - Button (primary, secondary, ghost, danger; sizes; icon slots)
  - Input Field
  - Select
  - Checkbox
  - Toggle
  - Feature Card
  - Stat
  - Badge
  - Alert
  - Avatar (with status indicator and group support)
  - Section Header
  - Navbar (with Logo)
  - Logo (mark and wordmark, light/dark)
- A landing page composition story demonstrating all components together.

## Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Storybook
